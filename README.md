# # Анализ рейтингов брендов
Проект для анализа рейтингов брендов на основе CSV-файлов. Скрипт читает один или несколько файлов с данными о товарах и формирует отчёт, например, средний рейтинг бренда.

## Пример запуска
установить зависимости 
pip install -r requirements.txt
выполнить скрипт
python main.py --files products1.csv products2.csv --report average-rating

### Добавление нового отчёта

Чтобы добавить новый отчёт:
1. Создайте класс в  reports/reports.py, наследуя BaseReport.
2. Реализуйте метод value(self, data), который принимает список словарей из CSV и возвращает список словарей с результатами.
3. Зарегистрируйте новый отчёт в словаре REPORTS в файле reports/__init__.py
