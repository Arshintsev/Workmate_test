# Анализ эффективности работы разработчиков
Проект для анализа эффективности разработчиков на основе CSV-файлов. 
Скрипт читает один или несколько файлов с данными о выполненных задачах и формирует отчёты о производительности сотрудников.

## Пример запуска
Установить зависимости 
```bash
    pip install -r requirements.txt
```
Выполнить скрипт
```bash
    python main.py --files employees1.csv employees2.csv --report average-performance
```
### Формат данных
CSV-файлы должны содержать следующие колонки:
name - имя сотрудника
position - должность
completed_tasks - количество выполненных задач
performance - показатель эффективности
skills - навыки и технологии
team - команда
experience_years - опыт работы в годах

#### Доступные отчёты
average-performance - средняя эффективность по должностям (отсортировано по убыванию)

### Добавление нового отчёта

Чтобы добавить новый отчёт:
1. Создайте класс в  reports/reports.py, наследуя BaseReport.
2. Реализуйте метод value(self, data), который принимает список словарей из CSV и возвращает список словарей с результатами.
3. Зарегистрируйте новый отчёт в словаре REPORTS в файле reports/__init__.py
